We Bare Capybaras

A fun tower defense game built with Unity

About the Project

We Bare Capybaras is a cute 2D tower defense game where capybaras defend their tower from invading enemies. The game features different hero types, enemy waves, and resource management. Built using Unity and C#, this project emphasizes game balancing and engaging animation.

Status: In Development

Technologies: Unity, C#
