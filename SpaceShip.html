Space Ship

A simple turn-based game built with SDL2 and C

About the Project

"Space Ship" is my first game developed when I just started university. It is a simple turn-based strategy game where players control a spaceship and fight against enemy ships and a boss. The game is built using SDL2 and C, without any advanced techniques. The core gameplay revolves around turn-based combat, where you need to plan your attacks carefully to defeat waves of enemies and powerful bosses.

Status: Complete

Technologies: SDL2, C

View Source Play Demo (Not Available)

Key Features

  • 🚀 Turn-based combat where the player and enemies take turns to attack.
  • 🌌 A variety of enemy ships, each with different abilities and behaviors.
  • 👾 Boss battles with unique patterns and greater difficulty.
  • 🔄 Object Pooling to manage game objects efficiently, such as enemies and projectiles.
  • 📝 Custom-built Linked List for storing and managing enemies in each turn.
